## 这是个什么？

此工具用于PwC的RA部门自动化处理一些CASE方面的问题，能很大程度上将我们的日常工作自动化。

[> 手动下载点击这里](https://github.com/mattholy/RA-AutomationTool/releases)

### 概要

此工具分为4个部分：

* RA Power Tool

* RA Power Addin

* JDT Enhanced Tool

* SQL Functions Set
